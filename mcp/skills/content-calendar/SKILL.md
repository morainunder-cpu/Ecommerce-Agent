# Content Calendar — 社媒内容日历

## 职责

为 Facebook HPD 和 TikTok @hpdun3 生成协调的内容发布计划，确保两个平台的内容节奏一致且互补。

## 平台策略

| 平台 | 账号 | 内容类型 | 发布频率 |
|------|------|---------|---------|
| Facebook | HPD | 产品展示 + 工厂实拍 + 客户案例 | 3-5 篇/周 |
| TikTok | @hpdun3 | 短视频（产品细节 + 生产过程 + 穿搭展示） | 1-2 条/天 |

## 内容矩阵

### Facebook 内容类型
- 产品上新帖（4 宫格图 + 规格文案）
- 工厂生产过程（Behind the scenes）
- 客户实拍/评价
- 行业资讯/工装知识
- 促销/活动帖

### TikTok 内容类型
- 产品细节短视频（15-30s，四宫格逐一展示）
- 生产过程快剪（15-60s）
- 员工穿搭展示
- 面料/工艺对比
- 热门挑战/趋势结合

## 日历模板

```
月度内容日历 — 2026年6月

W1（6/1-6/7）：
  周一 FB: 新品预告 | TT: 新品细节展示
  周三 FB: 工厂实拍 | TT: 生产过程快剪
  周五 FB: 客户案例 | TT: 穿搭展示
  
W2（6/8-6/14）：
  ...
```

## 处理步骤

1. 读取已完成的 Product Spec 列表
2. 确定产品上线优先级和时间线
3. 为每个产品生成 FB + TT 两条内容
4. 按周排列，避免同类型内容连续发布
5. 预留节日/促销内容位
6. 输出月度日历

## 与流水线联动

- 当 catalog-generator 完成一批产品 → 自动生成该批产品的社媒内容
- 引用 social-copywriter 生成的文案
- 日历输出到 Output/内容日历_{月份}.md
