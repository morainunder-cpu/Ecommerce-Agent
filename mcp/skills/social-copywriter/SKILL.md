# Social Copywriter — 社交媒体文案生成

## 职责
将产品数据转化为适用于抖音等社交平台的短视频脚本和营销文案。

## 输入
- Product Spec（来自 product-analyzer）
- 目标平台：抖音 / 微信视频号 / 小红书

## 文案类型

### 抖音短视频脚本
时长：15-30 秒
结构：
  0-3秒   钩子（产品亮点一句话抓住注意力）
  3-15秒  展示（四宫格细节逐一展示）
  15-25秒 卖点（材质、功能、使用场景）
  25-30秒 行动号召（引导购买/关注）

### 视频文案脚本
画外音 + 字幕：
  [场景描述]  [产品展示]  [细节特写]  [购买引导]
标签：#品类 #材质 #使用场景

## 风格参考
参照 舒特工贸抖音文案脚本库.txt 和 舒特工贸视频文案脚本库.txt 的已有风格：
- 口语化、亲切
- 突出产品核心卖点
- 节奏紧凑
- 适合工厂/工装品类

## 处理步骤
1. 从 Product Spec 提取核心卖点
2. 匹配目标平台选择文案类型
3. 参照已有脚本库风格生成文案
4. 自检：是否符合平台时长和内容规范

## 质量标准
- [ ] 核心卖点突出
- [ ] 时长符合平台规范
- [ ] 风格与已有脚本库一致
- [ ] 包含行动号召
