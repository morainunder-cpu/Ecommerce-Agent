# 商品图Prompt生成示例

## 场景

为"夏季工装夹克"生成Gem Image四宫格细节图Prompt。

## 输入

```
产品：夏季工装夹克
需求：
1. 反光条
2. 拉链
3. 袖口
4. 胸前口袋
比例：463:278
画质：4K
风格：电商摄影，自然光
```

## 输出（Gem Prompt）

```
**角色：** 资深电商AI提示词工程师
**产品：** 夏季工装夹克
**展示部位（四宫格细节图）：**
- 左上：反光条
- 左下：拉链
- 右上：袖口
- 右下：胸前口袋
**规格：** 463:278 | 4K | 电商摄影 | 自然光 | 真实材质
**约束：** 禁止logo、纽扣、口袋 | 禁止改变颜色、布料
```

## 验证清单

- [ ] 只展示描述中的部位
- [ ] 无logo
- [ ] 无纽扣
- [ ] 无多余口袋
- [ ] 颜色与产品一致
- [ ] 布料与产品一致
- [ ] 比例463:278
