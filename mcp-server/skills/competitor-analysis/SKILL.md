# Competitor Analysis — 竞品分析

## 职责

分析竞品产品的展示策略和卖点，为自身产品 Prompt 生成和目录制作提供差异化参考。

## 输入

- 竞品产品名称 + 描述（自由文本或链接）
- 自身产品 Product Spec（来自 product-analyzer）
- 分析目标（如：差异化卖点、定价参考、展示方式）

## 分析维度

### 1. 展示策略对比

| 维度 | 竞品 | 自身 | 差距 |
|------|------|------|------|
| 图片类型 | 四宫格/单图/场景图 | — | — |
| 展示部位 | 竞品突出部位 | 自身覆盖部位 | 缺失/冗余 |
| 拍摄风格 | 影棚/外景/白底 | 电商摄影，自然光 | 风格差异 |
| 文案策略 | 功能导向/情感导向 | 自身策略 | 可借鉴点 |

### 2. 卖点提取

从竞品描述中提取：

- 核心卖点（竞品主打的 1-2 个优势）
- 材质卖点（面料、工艺）
- 功能卖点（防水、透气、反光等）
- 视觉卖点（颜色、版型、细节）

### 3. 差异化建议

对比自身 Product Spec 与竞品后：

- 自身独有的优势 → 强化
- 竞品有自身无的 → 评估是否需要补充
- 双方共有的 → 差异化措辞

### 4. 定价与定位

- 竞品价格区间
- 竞品目标客群
- 自身定位建议（高端/性价比/差异化）

### 5. 违规检查

快速检查竞品展示中是否有违规元素：

- 是否有夸大宣传
- 是否有虚假材质描述
- 是否符合平台规则

## 处理步骤

1. 接收竞品信息 + 自身 Product Spec
2. 逐维度分析对比
3. 提炼可借鉴的展示策略
4. 输出差异化卖点建议
5. 标注需要警惕的竞品优势

## 输出格式

```
竞品分析报告

竞品：[名称]
来源：[链接/描述]

展示策略对比：
  图片类型：[竞品] vs [自身]
  重点部位：[竞品] vs [自身]
  拍摄风格：[竞品] vs [自身]

竞品核心卖点：
  1. [卖点]
  2. [卖点]

差异化建议：
  强化：[自身独有优势]
  考虑补充：[竞品有自身无]
  差异化措辞：[共有点的差异化方向]

定价参考：
  竞品价格：[区间]
  建议定位：[定位]
```

## 质量标准

- [ ] 竞品信息准确提取，无臆测
- [ ] 对比维度覆盖完整（展示/卖点/定价）
- [ ] 差异化建议可操作，非空泛
- [ ] 违规检查不遗漏明显问题
- [ ] 输出可直接用于 prompt_engineer 调整 Prompt 策略
